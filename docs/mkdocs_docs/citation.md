Citation
========

If you use RolyPoly in your research, please cite it as follows:

    Neri, U., Bushnell, B., Roux, S., & Camargo, A. P., Steindorff, A.S., Coclet, C., Parker, D., ... (2024). RolyPoly: A pipeline for RNA virus identification from RNA-seq data. [Software]. Available from https://code.jgi.doe.gov/UNeri/rolypoly

For specific versions or updates, please check the project repository
for the most current citation information.  
We hope to publish or have a preprint before mid 2025, which will be the offical v1.0. If you use RolyPoly before hand, we suggest mentioning the exact commit tag you used.
